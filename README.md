# zemptime-marketplace

Claude Code plugins by Chris Zempel.

## Installation

```
/plugin marketplace add zemptime/zemptime-marketplace
/plugin install vanilla-rails@zemptime-marketplace --scope user
```

## Plugins

### vanilla-rails

Production-tested Rails patterns extracted from Basecamp-style applications:

- **Controllers**: CRUD-only, resource extraction for state changes
- **Models**: Concerns as adjectives, state-as-records pattern
- **Testing**: Fixtures over factories, integration tests
- **Jobs**: Thin wrappers with `_later`/`_now` pattern
- **Hotwire**: Turbo Streams, focused Stimulus controllers
- **Style**: Method ordering, expanded conditionals
- **Work Breakdown**: Small focused PRs
